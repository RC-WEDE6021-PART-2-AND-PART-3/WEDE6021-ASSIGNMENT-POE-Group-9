PASTIMES PART 3 SETUP

1. Copy the Pastimes_Part3 folder to your XAMPP htdocs folder.
   Example: C:\xampp\htdocs\Pastimes_Part3

2. Start Apache and MySQL in XAMPP.

3. Open phpMyAdmin and make sure a database named clothingstore exists.
   (The SQL script also creates it automatically.)

4. Open this URL in your browser to create all tables and load sample data:
   http://localhost/Pastimes_Part3/loadClothingStore.php

FEATURES INCLUDED
- Customer browsing with search and filters
- Shopping cart with update/remove/continue shopping
- Checkout with delivery address capture
- Seller dashboard for sell requests with brand, description and image upload
- Admin user management (add, update, delete users)
- Admin clothing management (add, update, delete and approve clothing)
- Buyer/seller/admin messaging for delivery and item issues
- Order tracking for buyers and sellers
- Basic admin analytics cards
